## 📊 功能特色

### 1. 無前言 (No Frontmatter) 設計

與傳統的 Markdown 博客不同，這個系統不使用 frontmatter。所有的元數據都集中管理在 `metadata.ts` 文件中：

<CodeBlock 
  language="typescript" 
  filename="src/lib/metadata.ts"
>
{`export const blogPosts: Record<string, BlogPost> = {
  'welcome-to-my-blog': {
    slug: 'welcome-to-my-blog',
    title: '歡迎來到我的 Blog！',
    description: '這是我使用 Astro + React + MDX...',
    author: 'Ian',
    publishDate: new Date('2025-01-20'),
    tags: ['Astro', 'React', 'MDX', 'TypeScript'],
    featured: true,
    category: 'Tech',
    readingTime: 5,
    draft: false
  }
};`}
</CodeBlock>

### 2. TypeScript 支援

完整的 TypeScript 支援，包括：
- 型別安全的組件
- 介面定義
- 自動完成和錯誤檢查

### 3. 響應式設計

使用 Tailwind CSS 建立的響應式設計，在各種設備上都有良好的體驗。
